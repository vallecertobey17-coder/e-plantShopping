# coding-project-template

e-plantShopping is a modern, responsive e-commerce web application designed for plant enthusiasts. This application provides a seamless shopping experience, allowing users to browse a diverse catalog of indoor and outdoor plants, manage their selections, and proceed through a streamlined checkout process.

Built with React and managed with Redux Toolkit, the app demonstrates efficient state management and a component-based architecture to ensure a fast and interactive user interface.


Key Features
Curated Plant Gallery: Browse various categories of plants with detailed descriptions and pricing.

Real-time Shopping Cart: Add or remove items dynamically, with the cart automatically calculating total costs and quantities.

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

Efficient State Management: Utilizes Redux Slices to handle complex global states like cart updates and inventory.
